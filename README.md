# CPS Website

This is the website for CPS, which deals with finance, investment, money lending, and loans.

## Project Structure

```
cps-website/
│
├── index.html
├── about.html
├── services.html
├── contact.html
├── styles.css
├── script.js
└── README.md
```

## Instructions

1. Clone the repository.
2. Open `index.html` in your web browser to view the homepage.
3. Navigate to the other pages using the links in the navigation bar.

## Enhancements

- Added Bootstrap for better styling and layout.
- Implemented form validation using JavaScript.
- Improved the design and user experience.

## Contact

For any inquiries, please contact us through the form on the `contact.html` page.
